About Us

StayEase Rentals is a property booking platform designed to simplify reservation management and improve user experience.

This project was developed as part of ISM 6225 to demonstrate how a full booking system can be structured using modern web technologies.

Logical Data Model

The system is built on a structured data model representing relationships between properties, bookings, and customers. Key relationships include:

  • One property - many bookings
  • One customer - many bookings
