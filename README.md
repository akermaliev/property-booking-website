# 🏡 Property Booking Website

Static web application prototype for a property booking system developed as part of ISM 6225 – Application Development for Analytics.

## 🔗 Live Demo
- Website: http://myweb.usf.edu/~akermaliev/index.html  
- Chatbot: http://myweb.usf.edu/~akermaliev/mybot.html  

## 📌 Features
- Multi-page website with consistent layout and navigation
- CRUD simulation pages (Create, Read, Update, Delete)
- Interactive charts using Chart.js
- Embedded AI chatbot using Botpress
- Responsive design for different screen sizes

## 🤖 Chatbot
The chatbot is powered by Botpress and uses a structured knowledge base of property data.  
It allows users to:
- View available properties  
- Filter by location  
- Compare pricing  
- Ask general booking-related questions  

## 🧩 Data Model
The system is based on three core entities:
- User  
- Property  
- Booking  

Relationships:
- One User → Many Bookings  
- One Property → Many Bookings
